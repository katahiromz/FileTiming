**WHAT IS IT?**

    FileTiming is a command-line utility on file time for Windows.

    FileTiming can set file time and do statistics for files and folders.

    Written by Katayama Hirofumi MZ <katayama.hirofumi.mz@gmail.com>.


**SUPPORTED PLATFORMS**

    Windows 2000/XP/Vista/7


**USAGE**

    Usage: FileTiming [options] [directory]
    Options:
    -E                    Do stat for dates.
    -Y                    Do stat for years.
    -M                    Do stat for months.
    -D                    Do stat for day of months.
    -W                    Do stat for day of weeks.
    -T                    Do stat for time of days.
    -A                    Do stat for all information.
    -H                    Do also for hidden files/directories.
    -N                    Do set file time to now.
    -S YYYYMMDD           Do set file date (YYYYMMDD).
    -S YYYYMMDDHHmmss     Do set file date (YYYYMMDD) and time (HHmmss).
    -R YYYYMMDD-          Do set file date and time at random in the range.
    -R YYYYMMDD-YYYYMMDD  Do set file date and time at random in the range.
